# TOEFL 2026 — Listen & Repeat Study Pack

AI-assisted TOEFL iBT 2026 Listen & Repeat practice. Each lesson file contains 45 campus sentences grouped by theme, with a built-in AI prompt, pronunciation tips, and a 0–5 scoring rubric.

---

## How to Use with Gemini / ChatGPT / Claude

**Option A — Share the raw file URL (easiest)**

Copy the raw URL below and paste it into Gemini (or any AI with web access):

```
https://raw.githubusercontent.com/alpozturklive/toefl/main/lesson1.md
```

Then say: **"Read this file and start the Listen & Repeat session from Theme 1."**

The AI will read the built-in prompt inside the file and guide you sentence by sentence.

**Option B — Upload the file directly**

Download the `.md` file and upload it to Gemini / ChatGPT. Then say:
> "Start the Listen & Repeat session from Theme 1."

## What the AI Does Each Turn

1. Gives you **one sentence** at a time
2. You read it aloud (or type it)
3. AI gives feedback on **pronunciation, stress, thought groups**
4. Checks articles (a/an/the), final -s/-ed endings
5. Moves to the next sentence when you say **"Next"** or **"Hazırım"**

---

## Scoring (0–5)

| Score | Meaning |
|-------|---------|
| 5 | Fully intelligible, natural stress and intonation |
| 4 | Minor errors (article, preposition), meaning preserved |
| 3 | Some key words missing or changed |
| 2 | Most of the sentence missing, hard to understand |
| 1 | Only a few words, no sentence integrity |
| 0 | No response |
